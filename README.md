# 🛒 CodeAlpha Ecommerce Website

## 📌 Project Description

This is a Full Stack Ecommerce Website developed as part of the CodeAlpha Full Stack Development Internship.

The project allows users to browse products, register, login, add products to cart, checkout, and place orders. All data is stored in MongoDB.

---

## 🚀 Features

- User Registration
- User Login
- Product Listing
- Shopping Cart
- Checkout
- Order Placement
- MongoDB Database
- Express.js REST API

---

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

---

## 📂 Project Structure

```
backend/
frontend/
```

## ▶️ How to Run

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

Open `index.html` using Live Server in VS Code.
